## Descrição

Esta é uma mini API CRUD de usuários construída com NestJS. Ela oferece endpoints para cadastro, consulta, atualização de senha e remoção de usuários, além de autenticação via JWT para proteger rotas sensíveis.

## Instalação

```bash
npm install
```

## Como rodar o projeto

```bash
# desenvolvimento
npm run start

# modo watch
npm run start:dev

# produção
npm run start:prod
```

## Testes

```bash
# testes unitários
npm run test

# testes e2e
npm run test:e2e

# cobertura de testes
npm run test:cov
```

## Funcionalidades

- Cadastro de usuário
- Consulta de usuários (todos, por nome, por id)
- Atualização de senha
- Remoção de usuário
- Autenticação via JWT
- Proteção de rotas principais
